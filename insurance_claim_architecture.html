InsureCo Claim Submission Platform
Click any component above to inspect it, or run the end-to-end flow simulation below.
A policyholder files a claim through the React portal. The API Gateway authenticates the JWT, routes to the Claims Service which persists the claim and documents to PostgreSQL, then produces a claim.submitted event to Kafka. Downstream consumers handle adjudication, notifications, and fraud detection asynchronously.
